# Sudoku Puzzle Generator
Julius Lema

This project is meant to print out a properly solved Sudoku puzzle.
All you have to do is hit the Run.
By using an array of the first row, the rest of the numbers are printed using their indicies (the row and column they're at) and using that to take a number from the first row. By doing this, no two numbers are placed in the same row, column, or box.
